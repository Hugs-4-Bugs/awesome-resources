# Resumes Directory

Place your resume PDF files here.

## Required Files

- `resume_default.pdf` - Default resume (required)

## Optional Files

- `resume_java_backend.pdf` - Java Backend specific resume
- `resume_microservices.pdf` - Microservices specific resume

The system will automatically select the most appropriate resume based on the job posting.

## Note

Make sure your resume is in PDF format and named exactly as specified in `config.py`.
